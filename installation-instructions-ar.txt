
## 📦 كيفية التثبيت والتشغيل

```bash
# 1. حفظ السكريبت أعلاه في ملف
nano setup_silver.sh

# 2. إعطاء صلاحيات التنفيذ
chmod +x setup_silver.sh

# 3. تشغيل سكريبت التثبيت
./setup_silver.sh

# 4. الدخول إلى مجلد silver
cd silver

# 5. تثبيت المتطلبات
./install.sh

# 6. تشغيل الأداة
./run.sh https://example.com
